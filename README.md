# Linux Tinkering

The purpose of this repo is to keep all code related to any linux related
learning projects in one place. This will involve things like build environment
setup, basic kernel modules, etc. Work in this repo will primarily target the 
STM32MP157F-DK2 discovery kit but should be easily adaptable to any hardware.

Everything will be structured into sort of subprojects, meant to show a sort
of progression in developing for the kernel.
